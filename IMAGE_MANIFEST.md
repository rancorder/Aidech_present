# Image manifest

`images/` 配下に以下のファイルを配置します（リポジトリに同梱済み）。
スライド内容と画像の対応は `slides.json` の `image` フィールドと一致しています。

| Slide | Local file | Content |
|---:|---|---|
| 1 | `images/1.png` | 表紙 |
| 2 | `images/2.png` | こんなお悩み、ありませんか？ |
| 3 | `images/3.png` | 2つの支援 |
| 4 | `images/4.png` | ご相談から導入・伴走までの流れ |
| 5 | `images/5.png` | 導入によって期待できる変化 |
| 6 | `images/6.png` | サポート体制 |
| 7 | `images/7.png` | 集客の土台づくり |
| 8 | `images/8.png` | 3つの伴走プラン |
| 9 | `images/9.png` | 無料診断・ご相談 |
| 10 | `images/10.png` | 締め |

## 動画を入れる場合（任意）

同じ番号で `images/<N>.mp4` を置くと、そのスライドだけ自動で動画再生になります（例：`images/3.mp4`）。
ファイルが無ければ画像表示にフォールバックします。
